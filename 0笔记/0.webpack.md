# 常用插件、loader

1. webpack：
    `webpack webpack-cli`

2. 配置开发服务器：
    `webpack-dev-server`

3. 配置模板文件：
    `html-webpack-plugin`

4. 解析 .css文件 :
    `style-loader css-loader`

5. 解析 .scss文件 :
    `node-sass  sass-loader`

6. 单独提取 .css文件 :
    `mini-css-extract-plugin`

7. 解析图片（字体文件）：
    `file-loader url-loader`
    
9. 解析 ES6:
    `babel-loader @babel/core @babel/preset-env`  

10. 在HTML中引入图片：
    `html-loader`
